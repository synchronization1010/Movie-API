Movie API Documentation

Below is Documentation on how to utilize the Movie API. Please read carefully.

All data is returned as a JSON object.
Example:
{
title: "...",
year: "...",
genre: "...",
director: "...",
description: "..."
}

Request URL HTTP Method Response
Homepage / N/A Homepage.
Documentation /documentation N/A Documentation page.
Return a list of ALL movies to the user /movies GET JSON object holding data for all movies.
Return data about a single movie /movies/:title GET JSON object holding data for a specific movie.
Return data about a genre (description) by name/title. /genres/:genre GET JSON object holding data for movies of that ganre.
Return data about a director /directors/:directorName GET JSON object holding data for the director.
Register a new user /users POST JSON object holding data for newly registered user. (Required Input)
{
"Username" : "username",
"Password" : "password",
"Email" : "valid@email.com",
"Birthday" : "2010-01-01"
}
Allow users to update their user info (username) /users/:Username PUT Message indicating username was changed.
Add new movie to list of favorites /users/:username/movies/:movieID POST Message indicating movie was added successfully.
Remove movie from list of favorites /users/:username/movies/:movieID DELETE Message indicating movie was removed successfully.
Deregister existing user /users/:Username DELETE Message indicating the user was deleted.
